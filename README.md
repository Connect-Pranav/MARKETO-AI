## 🚀 Deploy to GitHub Pages

```bash
git clone https://github.com/[YOUR_USERNAME]/marketo-ai.git
cd marketo-ai
git add .
git commit -m "MARKETO AI v2.0 - Full platform"
git push origin main
```

Then go to: **Settings → Pages → Source → GitHub Actions**

Your URL: `https://[YOUR_USERNAME].github.io/marketo-ai`
